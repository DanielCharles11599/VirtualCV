# VirtualCV
This repository includes the code needed for the Virtual CV of Daniel Charles


1. Project Overview & Purpose
This repository contains the codebase for my Virtual CV, a dynamic and responsive portfolio website designed to showcase my professional identity, technical proficiency, and creative achievements in the IT field. This project serves as a key component of my professional portfolio, complementing my academic record at North-West University and my practical experience in web design and digital training.

Content: A dynamic portfolio designed to showcase my professional identity, skills, and achievements.
Value: This portfolio establishes my abilities in IT.


3. Detailed Elements of the Virtual CV:
* About Me: A professional summary highlighting my focus on AI integrations, data analytics, and full-system design.
* Projects: A gallery of my work with detailed descriptions and direct links to their respective GitHub repositories.* Skills: A summary of technical and professional proficiencies.
* Skills: A comprehensive list of my technical stack, including languages (Java, Python, C++, SQL) and frameworks (ASP.NET, WordPress).
* Certifications: Showcases the certifications that I have received.
* Testimonials: Provides recommendations and feedback that I have received from past employers.
* GitHub Dashboard: A real-time update of my activity on GitHub.


4. Project Functionalities:
* Modern Technologies: Throughout the entire project I have made extensive use of CSS and Javascript to implement animations, event listeners, and smooth transitions in order to make the website feel more modern.
* Responsive Design: Fully compatible with major web browsers and optimized for different devices (mobile, tablet, and desktop).
* Interactive Theme Toggle: A custom dark and light mode switcher that aligns with my professional brand identity.
* Dynamic Project Filtering: A JavaScript-powered filter allowing users to sort projects by tech stack (e.g., Java, Python, C#).
* CSS Animations: Subtle scroll-reveal and hover effects to enhance visual engagement without overwhelming content.
* Brand Assets: Includes a custom-designed favicon and logos that utilize scalable vector graphics for high-resolution displays.


5. Technical Stack:
* Frontend: HTML5, CSS3, JavaScript.
* Frameworks/Libraries: Integrated animations and responsive grid systems.
* Version Control: Managed via GitHub with a frequent commit history to demonstrate continuous development.
* Hosting: Deployed via GitHub Pages under the following URL: https://danielcharles11599.github.io/VirtualCV/.

   
6. Setup & Installation Instructions:
* To run this project locally for review or development, follow these steps:
  - Clone the Repository
  - Navigate to the Directory
  - Open the Project: Open index.html in any modern web browser (Chrome, Firefox, or Edge recommended).


7. Compatibility & Visual Standards
* The website was designed with useability in mind and includes multiple breakpoints to ensure a clean design on multiple sizes of device screens.
* The website includes a theme toggle to ensure pleasant viewing for all types of users.
* The website displays correctly on most major web browsers.
* The content was designed to provide a large amount of information without overwhelming the viewer.


8. Documentation & Branding
The visual style of this project is governed by the DC Brand Identity, utilizing a professional green and neutral palette as seen in the assets/logos/ directory.
